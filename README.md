# Setup & Installation

1. Clone the repository.
2. Install dependencies:
   
   `npm install`

    Install Playwright browsers:
   
   `npx playwright install`
   
3. Ensure your `skus.json` file is in the root directory.

# Run
Run `npm run dev`
After compeltion, open the generated ``product_data.csv``
To check for errors, open the generated ``errors.log``

To clean the above generated files run - `npm run cleanFiles`

# Test
Run `npm run test` to run the test
